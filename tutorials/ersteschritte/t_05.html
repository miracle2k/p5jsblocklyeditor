Tutorial: JavaScript generieren

Beispielprogramm starten:
JavaScript anzeigen lassen

  • Die Block-Programmierung ist die Vorzustufe zur Programmierung mit Programmiertext. Die Programmiersprache vom p5.js-Blockly-Editor ist JavaScript.

    Um aus den Blöcken JavaScript zu generieren, klickt man oben in der Mitte auf die Schaltfläche "JavaScript-Code".

    Es öffnet sich ein Fenster, in welchem das Programm in der Programmiersprache "JavaScript" angezeigt wird.

Das Programm im Text-Modus ausführen

  • Das mit Blöcken programmierte Programm kann mit einem Text-Editor geöffnet, bearbeitet und ausgeführt werden. Dazu klickt man oben in der Mitte auf die Schaltfläche "Text-Editor".

    Es wird ein neues Fenster geöffnet und der aus den Blöcken generierte JavaScript-Code wird angezeigt und kann ausgeführt werden.
