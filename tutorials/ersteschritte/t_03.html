Tutorial: Mit Blöcken arbeiten

Beispielprogramm starten:
Einen Kommentar einfügen

  • Ein Block kann mit einem Kommentar ergänzt werden. Ein Kommentar ist ein Informationstext und wird nicht ausgeführt.

    Klicke mit der rechten Maustaste auf einen Block und klicke dann mit der linken Maustaste auf "Kommentar hinzufügen".

    Klicke auf das Symbol "Fragezeichen" und tippe einen Kommentar ein.

    Einen Kommentar kann man verbergen und anzeigen, indem man wiederholt auf das Symbol "Fragezeichen" klickt.

Einen Block verschieben

  • Wenn man einen Block anklickt und mit gedrückter linker Maustaste verschiebt, werden dieser Block und alle Blöcke unterhalb verschoben.

  • Um nur einen einzigen Block zu verschieben, klickt man mit der linken Maustaste auf den Block um diesen zu markieren und drückt dann gleichzeitig die Tasten "Strg" und "x". Der Block wird dadurch ausgeschnitten.

    Um den Block wieder einzufügen, drückt man gleichzeitig die Tasten "Strg" und "v". Der Block erscheint wieder im Editor und man kann ihn an die gewünschte Position schieben.

Einen Block löschen

  • Um einen Block zu löschen, gibt es vier Möglichkeiten:

    1) Einen Block mit der linken Maustaste anklicken und dann die Tasten "Strg" und "x" gleichzeitig drücken.

    2) Einen Block mit der linken Maustaste anklicken und dann die Taste "Entf" drücken.

    3) Einen Block mit der rechten Maustaste anklicken und dann "x Bausteine löschen" anklicken.

    4) Einen Block anklicken und diesen Block und alle folgenden auf die Werkzeugleiste oder den Mülleimer schieben.

Einen Block wiederherstellen

  • Alle gelöschten Blöcke werden im Mülleimer zwischengespeichert.

    Um einen gelöschten Block wiederherzustellen, klickt man auf das Mülleimersymbol, so dass der Mülleimerinhalt angezeigt wird. Im Mülleimer klickt man auf einen Block, der dann in den Editor kopiert wird.

Einen Block kopieren

  • Um einen Block zu kopieren, gibt es zwei Möglichkeiten:

    1) Einen Block mit der linken Maustaste anklicken und dann die Tasten "Strg" und "c" gleichzeitig drücken. Danach die Tasten "Strg" und "v" gleichzeitig drücken und den Block an die gewünschte Position schieben.

    2) Einen Block mit der rechten Maustaste anklicken, dann "Kopieren" anklicken und den eingefügten Block an die gewünschte Position schieben.

Eine Aktion rückgängig machen

  • Die letzten Aktionen kann man rückgängig machen, indem man mit der rechten Maustaste auf einen leeren Editorbereich klickt und dann mit der linken Maustaste auf "Rückgängig" klickt.
