Tutorial: Das erste Programm

Beispielprogramm laden:
Trace-Tabelle erstellen

  • Nachdem das Programm geladen ist, kann man dieses durch Klicke auf "Nächster Schritt" starten und Schritt für Schritt ausführen.

    Überall dort, wo im Programm der Block "Trace-Eintrag dazu" eingefügt ist, wird der aktuelle Wert der in den Block eingebauten Variablen in der Trace-Tabelle ausgegeben.

    WICHTIG!
    Jeder Block "Trace-Eintrag dazu" muss einen einzigartigen Namen haben, damit die Trace-Tabelle richtig ausgefüllt wird.

    Wenn alle "Trace-Eintrag dazu"-Blöcke einen einzigartigen Namen haben, dann sollte die Trace-Tabelle fehlerfrei ausgegeben werden.

    Während das Programm Schritt für Schritt ausgeführt wird, kann der Editor mit der Maus verschoben werden oder gezoomt werden, ohne dass die Ausführung abgebrochen wird. Sobald man einen Block anklickt wird die Ausführung des Programms zurückgesetzt und die Trace-Tabelle gelöscht.
